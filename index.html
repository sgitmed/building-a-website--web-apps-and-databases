Web sites and Web Applications

What is a Web Application?

According to TechTarget, a web application is an application program that is stored on a remote server and delivered over the internet through a browser interface. This differs from websites. Where websites are typically used to display information, web Applications are designed to be interactive and have more functionality.

How do Web Applications interact with Databases?

Developers typically use databases to store and organize the data that their applications need. The web application interacts with the database by storing and retrieving data.

Use cases and examples:

Social Media

Social Media uses databases to store user profiles and information, comments, and likes. It also uses databases to generate content for users and manage connections.

Healthcare Management

Healthcare uses databases to manage patient records, appointments, and medical histories. It also is utilized for billing and insurance claims

E-Commerce

E-Commerce uses databases to manage product inventory, save user accounts, information, and orders.
